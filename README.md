# subscription_tracker

Subscription Tracker helps you take control of recurring expenses by showing all your subscriptions in one place, surfacing upcoming charges, and highlighting opportunities to save.

Overview
- Consolidates recurring services and payments into a single, easy-to-read dashboard.
- Shows total monthly spend, upcoming charges, and possible savings from underused subscriptions.
- Quick actions to add subscriptions, scan emails, and review recommendations.

Key Features
- Unified Dashboard: view total subscriptions and monthly cost at a glance.
- Upcoming Charges: timeline-style cards with due dates and amounts.
- Potential Savings: suggestions for services you rarely use.
- Privacy-first UI: local-focused UI and no unnecessary data sharing.

Quick Start
Prerequisites: Flutter SDK installed (see https://flutter.dev).

1. Clone the repo

```bash
git clone https://github.com/viktorapuyan/subscription_tracker.git
cd subscription_tracker
```

2. Install dependencies

```bash
flutter pub get
```

3. Run the app

```bash
flutter run
```

4. Run tests

```bash
flutter test
```
